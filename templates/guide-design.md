# ガイド用スタイルガイド — 「和風総本家」テーマ（正本）

`guide-template.html` の設計意図。**ガイド・デモ等のユーザー向け案内物はこのテーマ、
データレポート類は従来の `report-template.html`（デジタル庁準拠）** と使い分ける。

## 世界観

老舗の技術帳・番付表。職人の道具帳のように「静かで、確かで、手入れされている」印象を出す。
グラデーションヒーロー・絵文字・角丸カードの「AI生成っぽさ」を全排除する。

（CSS変数名 --ai/--shu は歴史的名称のまま、意味は抹茶/朱漆）

## 型（レイアウト語彙 — この語彙以外を発明しない。v2 京都風=「紋様ではなく原則」）

- **間（ま）が主役**: 表紙は 92vh、章間は 96〜150px。詰めて情報量を稼がない。空白は削る対象ではなく設計要素
- **表紙**: **縦書きの大題**（明朝 clamp 44-76px、letter-spacing .28em）を右に、リード文と丸落款を左下に置く非対称構図 + **日の丸**（薄紅 opacity .10 の大円。京風=面積は大きく色は淡く）+ 縦書きスクロール手掛かり「下へ」
- **和紙の質感**: feTurbulence ノイズ（SVG data URI）を地に敷く。画像読込なし
- **奥付帯**: 環境・機能・スキル・改訂を上下罫のみの帯で（枠で囲わない）。日付は「二〇二六年 文月」式
- **章見出し**: 漢数字 + 細罫 + 見出し + 右端に縦書き袖注。アラビア数字バッジ禁止
- **お品書き（言い方帳）**: 表組みでなく definition リスト（.oshina）。点線罫のみ、「言い方」は明朝 16.5px + 金茶の鉤括弧、効果は「─ 」で従属させる
- **心得**: 枠なし。抹茶の上罫 + 明朝見出しの三段組
- **工程図**: 縦罫で区切る飛び石式（.kotei、漢数字カウンタ内蔵）
- **注書き**: 朱漆の左罫 .chu、銘は縦持ちの小見出し
- **静かな出現**: IntersectionObserver で章をフェード（1s、translateY 14px）。表紙は墨入れの順（題→リード）。prefers-reduced-motion を必ず尊重

## 書体

- 見出し・言い方・ラベル: 明朝（Shippori/游明朝/ヒラギノ明朝 — システムフォントのみ、CDN禁止）
- 本文・表: ゴシック。行間 1.9、本文は 42em 以下に収める
- ラベルは letter-spacing .2〜.5em で「銘」らしく
