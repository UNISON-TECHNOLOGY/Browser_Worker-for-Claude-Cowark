# 日本市場SNS運用スキル

米国流ベストプラクティス（Hook→本文→CTA→ハッシュタグ）の構造を土台に、日本市場の実態（総務省調査・国内運用データ）で再編したもの。

## チャネル選定（日本の勢力図・全年代利用率）

| チャネル | 利用率 | 役割 | 注意 |
|---|---|---|---|
| LINE | 91% | **既存顧客CRM・再来訪**（米国のメール/SMS相当） | 攻めの新規獲得には不向き |
| YouTube | 81% | 検討層への深い説得・資産型コンテンツ | 制作コスト高 |
| Instagram | 53% | ビジュアル・世界観・採用ブランディング | 拡散力は弱い（保存・発見タブが鍵） |
| X | 43% | **拡散・リアルタイム・キャンペーン**（日本は匿名文化で米国より強い） | 炎上の主戦場でもある |
| TikTok | 33% | 若年層の新規認知（10-20代） | トーン設計必須（広告臭NG） |
| Facebook | 27% | 原則優先度低（中高年BtoBのみ検討） | 米国テンプレの FB 前提は無視してよい |

原則: 米国流の「FB/LinkedIn 偏重」配分は日本では X と LINE に振り替える。

## プラットフォーム別・投稿の型

**X**: 拡散はリポスト依存。ハッシュタグ 0〜2個（キャンペーンタグ除く）。「フォロー&リポストキャンペーン」は日本の定番形式。リアルタイム文脈（時事・季節）への乗り方が伸びを決める
**Instagram**: ハッシュタグ 3〜5個（詰め込みはリーチ制限リスク）。フィード=世界観、リール=新規到達、ストーリーズ=既存フォロワー接点。UGC リポスト運用が定番
**LINE公式**: **配信は週1回上限を基準**（毎日配信はブロック直行）。平均ブロック率約30%が相場 — 超過したら配信内容と獲得時の期待値のズレを疑う。配信=クーポン/限定情報など「受け取る理由」があるものだけ
**TikTok**: video-ad スキルの日本プリセット（UGC風・フルテロップ）に従う

## 投稿の構造（全チャネル共通の骨格）

1. フック（1行目で自分ごと化 — タイムラインでは1行目しか見えない）
2. 本文（1投稿1メッセージ、2〜4ポイント）
3. CTA（保存・プロフィールへ・リンクへ、のどれか1つ）
4. ハッシュタグ（上記のチャネル別個数）

## トーンの原則（日本仕様）

- **デフォルトは丁寧語+柔らかさ**。「中の人」型カジュアルはユーザーが明示的に選んだときのみ
- 炎上の影響は日本では特に重い（不買51%・採用応募回避77%の調査あり）。攻撃的・挑発的フック、政治・ジェンダー・災害に絡む時事乗りは公開前に必ず注意喚起する
- 第三者評価（口コミ・利用者数・認証）は日本で特に効く — ただし数値・No.1 表記は ad-compliance-jp のチェック必須

## 公開前チェック

- [ ] 1行目だけで興味を引けるか（フック）
- [ ] ハッシュタグ個数がチャネル相場内か
- [ ] トーンはデフォルト丁寧語か（カジュアルは指定があったときのみ）
- [ ] 時事・センシティブ文脈に触れていないか（触れるなら人間承認を明示要求）
- [ ] PR・タイアップなら「#PR」表記があるか → 詳細は ad-compliance-jp スキル
- [ ] LINE なら配信頻度が週1以内か
