# 動画広告クリエイティブスキル

プラットフォーム公式ガイド（TikTok Creative Best Practices / Google ABCD 等）の知見を再構成した、広告動画の企画・台本・絵コンテの作成規範。

## 出力形式: 絵コンテ表

台本は必ずこの表で出す（1行=1カット）:

| # | 秒 | 映像（カット内容） | テロップ | ナレーション/セリフ | 音 |
|---|-----|------|------|------|-----|

末尾に: 推奨尺 / アスペクト比 / セーフゾーン注意 / 差し替え可能なフック代案3本。

## 構成の型（黄金構造）

**0-3秒: フック → 3-15秒: 価値提案 → 15-25秒: 実証・社会的証明 → 最後5秒: CTA**

- **フック（0-3秒）が成否の8割**。3秒までに「自分ごと」と思わせる。6秒以内に主張を提示
- 冒頭にロゴだけ・社名だけは禁止（スキップされる）。ブランドは価値提案と同時に見せる
- 音声オフでも成立させる（テロップで完結）。ただし音ありを前提に設計（特にTikTok系）

## フックの型（代案出しに使う）

| 型 | 例の骨格 |
|---|---|
| 問いかけ | 「まだ◯◯してるの？」「◯◯で損してない？」 |
| 数字ショック | 「年収140万上がった話」「応募の9割が落ちる理由」 |
| Before見せ | 悩みの場面を3秒で再現→切り返し |
| 断言・逆張り | 「◯◯は間違い。」「◯◯、やめました」 |
| 当事者の一言 | 実際の利用者・社員の顔+一言（UGC風） |
| 途中から始める | 会話や作業の途中から入る（続きが気になる） |

## プラットフォーム別の型

| 項目 | TikTok/Reels/Shorts | YouTube インストリーム | Meta フィード |
|---|---|---|---|
| 推奨尺 | 21〜34秒 | 15秒 or 30秒 | 15秒以内 |
| 比率 | 9:16 縦 | 16:9（+縦併用） | 4:5 or 1:1 |
| フック猶予 | 3秒 | 5秒（スキップ前） | 3秒（無音再生前提） |
| トーン | UGC風・作り込みすぎない | ストーリー性可 | 静止画+動きでも可 |
| テロップ | 1画面5〜10語・大きく | 補助的 | 必須（無音前提） |
| CTA | 画面内+口頭 | 終盤+エンドカード | ボタンと重複させる |

**YouTube は ABCD で検収**: Attention（冒頭で掴む）/ Branding（早期に自然に）/ Connection（感情・自分ごと）/ Direction（明確な次の行動）。

## LP/商材からの展開手順（/delve-adlp の逆方向）

1. LP のトークン・訴求（knowledge/styles/ にあれば再利用）から、**最強の1訴求**を選ぶ（全部入り動画は禁止 — 1動画1メッセージ）
2. 訴求タイプに合わせてフック型を選定し、代案3本
3. 黄金構造に流し込み、プラットフォーム別に尺・比率・テロップ量を調整
4. 実証パートには LP の実績数値・お客様の声をそのまま使う（捏造禁止、ないものは `[要確認]`）
5. CTA は LP の主CTA文言と一致させる（メッセージマッチ）

## 日本市場プリセット（デフォルト適用）

編集定数は skills/web-design/resources/jp-taste.md §5 に従う（カット1-3秒・白字黒縁テロップ・中央帯セーフゾーン・ループ構成・軽いオチ）。

日本向け出稿では以下を標準とする（米国流との差分）:

- **フルテロップ必須**: 日本は字幕文化（テレビバラエティ由来）+音声オフ視聴前提。全セリフ・ナレーションにテロップを乗せる
- **UGC風・非広告感を優先**: 作り込んだプロモ映像より「素のまま」感が効く。Z世代は「盛らない」文化（無加工・実物感）。過度な演出・誇張ライティングは逆効果
- **日本で効くフックの型を優先**: あるある系（悩みの再現）/ 検証系（本当に◯◯なのか試した）/ 開封・体験系 / ビフォーアフター（※効果断定は ad-compliance-jp 参照）
- **挑発的フックは抑制**: 米国流の大胆な断言・煽りは日本では炎上リスク過大。断言型を使う場合は自虐・自社方針の宣言（「大量応募、しません」等）に限定
- **法規チェック必須**: 完成した台本は ad-compliance-jp スキルで検査（No.1表記・効果断定・PR表記・求人系の条件明示）

## 検収チェック

- [ ] 3秒時点で「誰向けの何の話か」分かるか
- [ ] 音声オフで意味が通るか（テロップ完結）
- [ ] 1動画1メッセージか（訴求の詰め込みがないか）
- [ ] 実証（数値・声・デモ）が入っているか
- [ ] CTA が具体的な行動+LP と同一文言か
- [ ] 尺・比率・テロップ量が出稿先の型に合っているか
- [ ] フック代案が3本あるか（ABテスト前提）
