# セールスライティング スキル
**対象領域: 無形商材 × 人材業界**

人材紹介・人材派遣・採用支援などの無形サービスの価値を伝え、企業（クライアント）や求職者の行動を促す文章作成技法。

## Use this skill when
- クライアント企業への提案書・企画書を作成するとき
- 新規開拓のテレアポスクリプトを作成するとき
- 商談後のフォロー・クロージングメールを作成するとき
- サービスLP（BtoB/toC）の本文構成を求められたとき

## Do not use this skill when
- 社内メールや事務連絡を求められたとき（-> business-writing）
- 求人票の作成を求められたとき（-> recruit-writing）
- キャッチコピーだけを求められたとき（-> copywriting）
- データ中心のレポートを求められたとき（-> logical-writing）

## 無形商材セールスの最大の課題

> **見えない、触れない、試せない → 信頼がすべて**

### 有形 vs 無形の違い

| | 有形商材 | 無形商材（人材） |
|---|---|---|
| 購入前 | 実物を見れる | 見えない |
| 判断基準 | スペック比較 | 信頼・実績 |
| 効果実感 | 即時 | 時間がかかる |
| 失敗リスク | 返品可能 | 採用失敗=大損失 |

→ **だから「信頼」と「具体性」で売る**

## セールスフォーミュラ（人材業界版）

### PAS（Problem → Agitate → Solve）
```markdown
【Problem: 採用の痛みを言語化】
「採用に3ヶ月、100万円かけて、3ヶ月で辞められた経験はありませんか？」

【Agitate: 痛みを掘り下げる】
「現場は疲弊し、残されたメンバーのモチベーションも低下。
 採用コストだけでなく、教育コスト、機会損失...
 "合わない採用"の本当のコストは、年収の3倍とも言われています。」

【Solve: 解決策を提示】
「[サービス名]は、スキルだけでなくカルチャーフィットまで見極める
 独自の評価手法で、定着率92%を実現しています。」
```

### BAB（Before → After → Bridge）
```markdown
【Before: クライアントの現状】
「求人を出しても応募が来ない。来ても求めるレベルに達していない。
 面接に時間を取られ、本業に集中できない日々...」

【After: 理想の未来】
「厳選された3名の候補者から選ぶだけ。
 面接は最終1回のみ。空いた時間は事業に集中。」

【Bridge: サービスが橋渡し】
「[サービス名]が、母集団形成から一次面接まですべて代行します。」
```

## BtoB特化: 法人営業（RA）向けスクリプト・メール

### 新規開拓テレアポスクリプト（受付突破〜担当者接続）
```markdown
【受付突破】
「[自社名]の[名前]と申します。
 御社のエンジニア採用の件で、30%ほど採用単価を抑える新しい手法について
 人事責任者様にお伝えしたいことがありお電話しました。お取次ぎいただけますか？」

【担当者接続後・フック（15秒）】
「突然のお電話失礼いたします。
 現在、多くのIT企業様で『求人媒体を出しても応募が来ない』というお悩みを伺います。
 御社でも、特にインフラエンジニアの採用で苦戦されていませんでしょうか？」
 （※相手の「痛み」に直球で触れ、Yes/Noで答えられる質問を投げる）

【切り返し（費用対効果を疑われた時）】
「おっしゃる通り、初期費用は導入のネックになりやすいです。
 ですから弊社は完全成果報酬型です。万が一採用できなかった場合、
 御社から頂く金額は1円もございません。まずは情報交換だけでもいかがですか？」
```

### 商談後フォローメール（クロージング）
```markdown
件名: 本日のお打ち合わせの御礼および、ご提案書の送付

[担当者名]様

本日は貴重なお時間をいただき、誠にありがとうございました。
御社の「営業部門の離職率」に関する課題について、深く理解いたしました。

■ 弊社がお力になれる理由
1. 定着率92%: 単なるマッチングではなく、カルチャーフィットを最重視します。
2. 平均紹介スピード[X]日: 欠員補充の急務にもスピーディに対応可能です。

添付にて、本日お話しした「[職種名]特化型 採用支援プラン」の資料をお送りします。
ご検討いただき、来週[曜日]までに一度フィードバックをいただけますと幸いです。
```

## 提案書テンプレート（対クライアント企業）

```markdown
## 1. エグゼクティブサマリー
[1段落: 何を提案し、どんな採用成果が見込めるか]

## 2. 貴社の採用課題（ヒアリング内容の整理）
- 課題1: [具体的に。数字で]
- 課題2: [具体的に。数字で]

## 3. 解決策
### サービス概要
[何をどう提供するか]

### なぜ当社か（競合との違い）
| | 当社 | 一般的なエージェント |
|---|---|---|
| 候補者の質 | 独自DB+ヘッドハント | 求人媒体からの応募者 |
| 定着率 | 92%（1年後） | 業界平均70% |
| 紹介スピード | 平均5営業日 | 平均2-3週間 |

### 期待効果
- 採用コスト: 現状[X万円] → [Y万円]に削減見込み
- 採用期間: [X週] → [Y週]に短縮
- 定着率: [X]% → [Y]%

## 4. 料金体系
※成果報酬型: 入社時に年収の[X]%
※返金保証: [X]ヶ月以内の退職は[X]%返金

## 5. 導入事例
### [同業界の企業名]
- 課題: [Before]
- 施策: [何をしたか]
- 結果: [数字で]

## 6. 次のステップ
1. 週1回の定例MTG開始
2. 求人要件の確定（3営業日以内）
3. 候補者紹介開始（1週間以内）
```

## LP構成（人材サービス向け）

### 対企業（BtoB）
```
1. ヘッド: 採用の痛みに共感するキャッチ
2. 課題提示: 3つの採用あるある
3. 解決策: サービスの仕組み（図解）
4. 実績: 数字（導入社数、定着率、紹介スピード）
5. 事例: Before/After（同業界2-3社）
6. 料金: 成果報酬型で初期費用ゼロを強調
7. FAQ: 「本当に質の高い候補者が来るの？」に答える
8. CTA: 「まずは無料相談（30分）」
```

### 対求職者（toC）
```
1. ヘッド: 転職の不安に寄り添うキャッチ
2. 共感: 「こんな悩みありませんか？」3つ
3. サービス紹介: 何をしてくれるのか（図解）
4. 実績: 転職成功者の声（実名+写真+数字）
5. 流れ: 登録→面談→紹介→面接→内定（ステップ図）
6. Q＆A: 「本当に無料？」「今すぐ転職しなくてもOK？」
7. CTA: 「無料キャリア相談に申し込む」
```

## 信頼構築パーツ（無形商材の生命線）

| パーツ | 人材業界での例 |
|---|---|
| 数字実績 | 「紹介実績3,200名、取引企業500社」 |
| 顧客の声 | 企業人事と転職者、両方の声を掲載 |
| Before/After | 「採用期間3ヶ月→2週間に短縮」 |
| リスクリバーサル | 「入社3ヶ月以内の退職は全額返金」 |
| プロセス開示 | 候補者選定の基準を公開（ブラックボックスにしない） |
