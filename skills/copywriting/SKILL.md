# コピーライティング スキル
**対象領域: 無形商材 × 人材業界（toC）**

形のないサービスの価値を、短い言葉で可視化する。求職者・転職者・企業の心を掴み、行動を促すコピーの作成技法。

## Use this skill when
- キャッチコピー・タグラインの作成を求められたとき
- 求人バナーやSNS投稿用の短文コピーが必要なとき
- スカウトメールの件名（13文字以内）を求められたとき
- LP（ランディングページ）の見出しを求められたとき

## Do not use this skill when
- 求人票の全体構成（仕事内容・募集要項含む）を求められたとき（-> recruit-writing）
- 長文のストーリー記事を求められたとき（-> storytelling）
- BtoB向けの提案書やメールを求められたとき（-> sales-writing）

## 🤖 System Prompt（AIへの指示前提）
```text
あなたは人材業界に特化したトップクラスのコピーライターです。
無形商材の抽象的な魅力を、求職者や企業の心が動く「具体的で短い言葉」に変換するスキルを持っています。
出力するコピーは最大でも13文字以内、最強は7文字であるという鉄則を遵守してください。
```

## コピーの2大分類

### イメージコピー（ブランディング目的）
- **目的**: 企業・サービスの世界観・価値観を伝える。直接売らない
- **効果**: 記憶に残る、好感度を上げる、「この会社で働きたい」を醸成
- **人材業界での使いどころ**: 採用ブランディング、企業理念、タグライン

```
「はたらく、を変える。」（9文字）
「あなたの"好き"が仕事になる。」（13文字）
「人が、未来をつくる。」（9文字）
「出会いを、チカラに。」（9文字）
「可能性は、無限大。」（8文字）
```

### セールスコピー（応募・問い合わせ喚起）
- **目的**: 今すぐ応募・登録・問い合わせさせる
- **効果**: エントリー数、登録率、CVR直結
- **人材業界での使いどころ**: 求人バナー、LP、スカウトメール件名

```
「年収100万アップ、実現。」
「未経験OK、月給35万〜。」
「まず話だけでも。」
```

## 文字数の鉄則

> **最強は7文字。最大でも13文字。**

| 文字数 | 評価 | 理由 |
|---|---|---|
| **7文字** | ★★★ 最強 | 一目で入る、記憶に残る |
| 8〜10文字 | ★★☆ 良い | 十分短い |
| 11〜13文字 | ★☆☆ 許容 | これが限界 |
| 14文字以上 | ☆☆☆ NG | 削り直す |

## 無形商材コピーの最大の壁

> **「形がない = 想像できない = 価値がわからない」**

### 解決策: 抽象を具体に変換する

| 無形のまま（NG） | 具体化（OK） |
|---|---|
| 「キャリアサポート」 | 「年収、80万上がった。」 |
| 「成長できる環境」 | 「入社1年で管理職。」 |
| 「充実の研修制度」 | 「3ヶ月で一人前。」 |
| 「働きやすい職場」 | 「18時、全員帰宅。」 |
| 「やりがいのある仕事」 | 「"ありがとう"が日常。」 |

## コピー作成フレームワーク（人材特化）

### 型1: 体言止め（最短・最強）
```
「次のキャリア。」（7文字）
「はたらく幸せ。」（7文字）
「本気の転職。」（6文字）
```

### 型2: Before → After（変化を見せる）
無形商材は「変化」が最大の価値。
```
「迷いが、確信に変わった。」（11文字）
「"嫌い"が"好き"に変わる仕事。」（13文字）
「日曜が怖くなくなった。」（11文字）
```

### 型3: 求職者の本音を代弁
```
「もう我慢しなくていい。」（11文字）
「このままでいいの？」（9文字）
「好きなことで、生きる。」（10文字）
```

### 型4: 数字+ベネフィット
人材業界で刺さる数字 = 年収、時間、人数
```
「年収+150万の転職。」（10文字）
「残業ゼロ、年収UP。」（9文字）
「転職成功率98.2%。」（10文字）
```

### 型5: 呼びかけ型（ターゲット直撃）
```
「30代、本気の転職へ。」（10文字）
「ママの再出発。」（7文字）
「未経験のあなたへ。」（9文字）
```

### 型6: 断言・宣言型
```
「人生は、変えられる。」（9文字）
「仕事を選ぶ時代。」（8文字）
「妥協しない。」（6文字）
```

## ブラッシュアップ手法

### Step1: 書き出す（20案以上、質より量）
### Step2: 13文字以下に削る
```
Before: 「あなたのキャリアに寄り添い最適な提案をします」（21文字）
After:  「寄り添う転職。」（7文字）
```

### Step3: "形のない価値"を五感に変換する
```
Before: 「やりがいのある仕事」
After:  「鳥肌が立つ仕事。」（8文字）
```

### Step4: 音読してリズム確認
- 七五調で整える
- 句読点「、」「。」でリズムを作る

## コピー評価チェックリスト

- [ ] **13文字以内か？**（7文字なら最高）
- [ ] **無形の価値が"見える化"されているか？**
- [ ] **求職者/企業が"自分ごと"と感じるか？**
- [ ] **声に出して気持ちいいリズムか？**
- [ ] **イメージ or セールス、目的に合っているか？**
- [ ] **同業他社と差別化できているか？**

## 媒体別最適化ガイド（人材業界toC）

| 媒体 | 文字数 | 構成ポイント | CVR向上の鍵 |
|---|---|---|---|
| 企業タグライン | 7文字 | 採用ブランドの本質 | 全媒体で統一使用 |
| 求人サイト見出し | 7〜13文字 | 数字+ベネフィット | 検索結果一覧で目立つこと |
| 求人バナー（GDN等） | 7文字 | 数字+変化、視認性重視 | 着地LPとトーンを揃える |
| スカウトメール件名 | 〜13文字 | パーソナル感、名前・経歴に言及 | 開封率が応募率を左右 |
| X（旧Twitter） | 7〜10文字+絵文字 | 共感・本音代弁型が強い | リプ・引用RTで拡散→LP誘導 |
| Instagram/Reels | 7文字（テキスト少なめ） | ビジュアル主体、テキストは補助 | プロフィールリンクへの動線 |
| LinkedIn | 13〜20文字（長め可） | 専門性・実績ベース | 「なぜあなたに」の理由を添える |
| LP見出し | 7〜13文字 | 痛みor憧れ、スクロール誘導 | ファーストビューで離脱を防ぐ |

### 媒体別の注意点
- **求人サイト**: 他社求人と並ぶため「数字の具体性」が差別化の鍵
- **SNS（X/Instagram）**: 7文字ルールが最も活きる。短さ=スクロール停止力
- **LinkedIn**: 唯一13文字を超えてよい媒体。プロフェッショナル文脈では具体性>短さ
- **バナー広告**: クリック後のLPとコピーのトーンが乖離すると離脱率が跳ね上がる
