# モックアップ/LP 出荷前チェックリスト

design-critic の審査項目、および Delvework の Verify ステップで使う。Front-End-Checklist 系の公開知見から LP 制作に必要な項目を再構成。

## Critical（1つでも該当したら REVISE）

- [ ] ダミーテキスト・ダミー画像が残っている
- [ ] 主CTAがファーストビューに存在しない
- [ ] アクセント色がCTA以外に使われている（視覚的希少性の毀損）
- [ ] 本文コントラストが 4.5:1 未満
- [ ] 375px幅で横スクロールが発生する / カードが畳まれない
- [ ] `<title>` / meta description / OGP がない
- [ ] 画像に alt がない（装飾画像は alt="" 明示）
- [ ] 課題診断の high 項目が未解消

## High

- [ ] 見出し階層が h1→h2→h3 の順序を守っていない
- [ ] フォーム項目が5個以上ある / ラベルが placeholder 頼み
- [ ] hover / focus 状態が未実装（キーボード操作で現在地が見えない）
- [ ] 余白が8pxグリッド外の値 / セクション間隔が不均一
- [ ] フォントが3種以上 / タイポスケール外のサイズがある
- [ ] 数値訴求に比較対象・期間・単位がない

## Medium

- [ ] 画像に width/height 未指定（CLS要因）
- [ ] 1行の文字数が全角45字を超える
- [ ] リンクテキストが「こちら」（文脈で行き先が分かること）
- [ ] アニメーションが意味を持たない装飾 / 300ms超
- [ ] print で崩れる（レポート系のみ）
