# 日本広告法規チェックスキル

生成した広告・求人表現が行政処分リスクを持たないかの公開前検査。**これは品質チェックではなく法的ゲート** — 該当項目は「直すか、根拠を用意するか、人間の法務確認に回すか」のいずれかで必ず処理する。

## 1. 景品表示法（全広告に適用）

| 検査項目 | ルール |
|---|---|
| No.1・最大級表現（「No.1」「日本一」「業界最大」「顧客満足度1位」） | **客観的調査の裏付け+調査機関・調査時期・出典の併記が必須**。イメージ調査ベースのNo.1は措置命令事例あり。根拠が確認できなければ使用不可 |
| 優良誤認 | 実際より著しく優れていると誤認させる表現の禁止（効果の断定・誇張、加工画像による性能偽装） |
| 有利誤認 | 二重価格（「通常◯円→今だけ◯円」）は通常価格の販売実績が必要。「期間限定」の常態化は不可 |
| 打消し表示 | 「※個人の感想です」等は本文と同等の視認性が必要（極小文字での免責は無効） |

## 2. ステマ規制（2023年10月〜、処分対象は広告主企業）

- 事業者が関与する表示（インフルエンサー投稿・レビュー依頼・ギフティング）には **「#PR」「#広告」等の明示が必須**
- 第三者を装った自作レビュー・依頼レビューは不当表示
- 生成する投稿がタイアップ/依頼投稿の文脈なら、PR表記をデフォルトで入れる（外すのは人間の判断のみ）

## 3. 薬機法（該当領域のみ: 化粧品・健康食品・医療機器・美容系）

- 「治る」「痩せる」「若返る」「シミが消える」等の効能効果の断定は原則不可
- 化粧品は認められた効能56項目の範囲内でのみ表現可
- 該当領域と判定したら: 断定表現を「うるおいを与える」等の許容範囲へ書き換え+「薬機法領域のため法務確認推奨」と明記

## 4. 職業安定法（求人広告・スカウト・採用ページ）

- **的確表示義務**: 虚偽・誤解を生む表示の禁止、情報を正確かつ最新に保つ義務
- **労働条件の明示**: 賃金（幅がある場合は幅と根拠）・勤務地・業務内容・雇用形態・試用期間の記載
- 「みなし残業」を含む場合は時間数と超過分の扱いを明示
- モデル年収を出す場合は前提（年次・実績）を併記。最高値だけの提示は誇張リスク
- 実在しない好条件求人（釣り求人）は禁止

## 5. 実行手順

1. 成果物のコピーを走査し、上記の該当パターンを列挙する
2. 各該当項目を3分類で処理:
   - **修正**: 表現の書き換えで解消できるもの（断定→範囲内表現等）
   - **根拠要求**: `[要根拠: ◯◯]` を付けてユーザーに事実確認を求めるもの（No.1・実績数値・価格）
   - **法務確認**: 薬機法領域・際どい比較広告は「人間の法務確認推奨」と明記
3. チェック結果を成果物の末尾に「法規チェック: 問題なし / 修正◯件 / 要確認◯件」として1行添付

## 免責

このチェックは網羅的な法務レビューの代替ではない。最終判断は人間（必要に応じて専門家）が行うこと。
