# 部品: SNSリサーチ（トレンド・競合アカウント調査）

対象媒体のパック手順（procedures/delve-sns-*.md）の「見える範囲・制約」を先に確認し、その範囲で調査する。

## 手順
1. 調査目的を確定（トレンド俯瞰 / 競合アカウント分析 / 個別投稿の深掘り）
2. 媒体別の最適ルートで読む: note=全文読取 / YouTube=字幕パネルで文字起こし / X=数値+リプ欄 / Instagram=キャプション+サムネ / TikTok=Creative Center 優先
3. 動画の深掘りはフレーム抜き取り（音声不可）。全編精読が必要ならユーザーに専用DLフォルダ保存を依頼 → video-edit.md のフレーム分解
4. 結果は knowledge/sns/<媒体>/research/ に記録し、後段のクリエイティブ（投稿作成）に接続
5. bot検知・CAPTCHA・レート制限に遭遇したら即中断して報告（突破しない）
