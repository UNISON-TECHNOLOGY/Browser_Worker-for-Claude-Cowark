# 行動経済・ナッジ判断スキル（日本実証ベース）

日本の実証研究（環境省ナッジ・ユニットBEST、厚労省受診勧奨、大竹文雄・佐々木周作らのRCT）から蒸留した**フレーム選択の判断表**。数値は実測。

## 大原則（これだけは常に）

1. **最大の効果量は文言でなく「簡素化・デフォルト」** — 環境省の国内最大RCT（30万世帯）でも安定効果は行動の具体化・簡易化。コピーをいじる前に手数を削れ（入力項目削減・返信だけでOK化・日程候補提示済み）
2. **単発CV獲得＝損失フレーム / 関係維持＝利得フレーム** — 損失フレームは単発行動に強い（八王子市大腸がん検診: 損失29.9% vs 利得22.7%、+7.2pt）が、意向だけ上げて実行動につながらず心理負担を残す（大竹）。メルマガ・採用ブランディング等の継続接点は利得基調
3. **効果は数%ポイント刻みが現実** — 出版バイアス補正後のナッジ平均効果はほぼゼロの可能性（PNAS 2022）。「心理テクだから効く」を前提にせず、cro-jp の前後比較/ABで都度計測

## フレーム選択表

| 状況 | 使うフレーム | 型 | 根拠 |
|---|---|---|---|
| 締切のある単発行動（応募・イベント・検診型） | 損失回避 | 「〜できる権利がなくなります」「◯日までなら書類選考免除」 | 八王子+7.2pt / スカウト返信率3.4%→6%の実務例 |
| 継続関係（採用広報・メルマガ・ファン化） | 利得 | 「〜が得られます」 | 大竹: 利得は効果が長期持続 |
| 肯定的な層の後押し | 社会規範（実数つき） | 「同年代の◯%がすでに」「同じ大学出身の先輩◯名が活躍」 | 高知市受診勧奨で改善 |
| 反発が予想される依頼 | 利他 | 「あなたの◯◯がチーム/顧客の◯◯を守る」 | COVID実験で実行動を促進（短期） |
| リスク回避が強い読み手（日本の平均） | 現状維持の損失への転換 | 「動くリスク」でなく「動かないことで失う市場価値・選択肢」 | 日本人の損失回避は国際的に強い（ハイリスク投資同意12% vs 他国30%超） |

## 禁忌表（使うと逆効果 or 違法リスク）

| 禁忌 | 理由 |
|---|---|
| **否定的な層への社会規範訴求** | バックファイア実証（ワクチン実験: 非希望者には逆効果） |
| **若年層（25-34歳）への汎用ナッジ** | 同実験で全メッセージ無効。若手に効く唯一の変数は「候補者固有の個別具体」（テンプレ臭が最大の返信率キラー） |
| 抽象的なお願い（「ご協力ください」） | タオル実験で協力率-5%。数字つき記述的規範に置き換える |
| 「平均より下」を突きつける比較 | ブーメラン効果。「あなたは上位◯%」型の承認と併用する |
| **偽の希少性・偽カウントダウン・実数でない「◯人閲覧中」** | 消費者庁ダークパターン規制（2025実態調査、法改正検討中）。数字は実数のみ |
| 不安煽り一辺倒 | 精神的負担→ブランド毀損（損失フレームのスラッジ化）。断る自由を明示する（「情報収集目的の返信も歓迎」） |
| 有料オプションのデフォルトON | ダークパターン類型（sneaking） |
| 「一部の人だけ得をする」見せ方 | 日本人のスパイト（妬み回避）傾向。「あなたも同じ条件で」の公平感を添える |

## レビュー用チェックリスト（EAST — 生成後にこの4点で自己審査）

- **E**asy: 読み手の手数は最小か（返信だけ/1タップ/入力ゼロに近いか）
- **A**ttractive: 件名・冒頭は「その人固有」か（汎用の美文より個別の事実）
- **S**ocial: 規範の数字は実数か。否定的な層・若年層に向けていないか
- **T**imely: タイミングは合っているか（転職なら更新時期・賞与後、検討の入口か出口か）
