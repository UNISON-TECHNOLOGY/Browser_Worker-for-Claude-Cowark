# ビジネスライティング スキル
**対象領域: 無形商材 × 人材業界**

人材ビジネスにおける社内外のコミュニケーション文書。クライアント企業への報告、求職者への連絡、社内共有など。

## Use this skill when
- クライアント企業へのメール（ヒアリング依頼・日程調整・選考結果通知等）を作成するとき
- 求職者への連絡メール（面接日程・選考結果等）を作成するとき
- 社内共有文書（候補者紹介レポート等）を作成するとき
- ビジネスメールのトーン・マナーを整えるとき

## Do not use this skill when
- セールス目的の提案書・テレアポスクリプトを求められたとき（-> sales-writing）
- 求人票や採用コンテンツを求められたとき（-> recruit-writing）
- データ分析や論理的な報告書を求められたとき（-> logical-writing）

## 🤖 System Prompt（AIへの指示前提）
```text

論理的でありながら、人を不快にさせない温かみとプロフェッショナルさを兼ね備えたビジネスメール・文書を作成します。
```

## 基本原則

### 1. 結論ファースト（PREP法）
- **Point**: 結論を最初に
- **Reason**: 理由を述べる
- **Example**: 具体例で補強
- **Point**: 結論を再度

### 2. 簡潔性
- 1文60文字以内
- 冗長表現の排除（「〜させていただく」→「〜します」）
- 箇条書きの活用

### 3. 相手別の温度感
- 人材業界は「人」が商材。冷たい文面は致命的
- 事務的すぎず、馴れ馴れしすぎないバランス

## 人材業界 文書タイプ別テンプレート

### クライアント企業への求人ヒアリング依頼
```
件名: 【ご確認】〇〇職の求人要件について

〇〇株式会社 △△様

お世話になっております。[会社名]の[名前]です。
〇〇職の募集について、求人要件を確定させたくご連絡いたしました。

■ 確認したいポイント
1. 必須スキル/経験（年数の目安も）
2. 想定年収レンジ
3. 入社希望時期
4. 選考フロー（面接回数など）

■ 次のステップ
ご回答いただき次第、候補者のサーチを開始いたします。
目安として1週間以内に3名以上ご紹介する予定です。

ご不明点があればお気軽にどうぞ。
```

### 求職者への面接日程調整
```
件名: 【面接日程のご相談】〇〇株式会社 △△職

[名前]様

お世話になっております。[会社名]の[担当者]です。
[企業名]の[職種]ポジションについて、面接日程のご相談です。

■ 候補日時（いずれも所要30分程度）
① X月X日（月）14:00〜
② X月X日（水）10:00〜
③ X月X日（金）16:00〜

■ 面接形式: オンライン（Zoom）
■ 面接官: [役職・名前]

上記でご都合の良い日時をお知らせください。
別日程のご希望があれば調整いたします。

ご面接に向けて、企業情報や想定質問もお伝えできますので
お気軽にご相談ください。
```

### 選考結果の通知（不採用の場合）
```
[名前]様

お世話になっております。[会社名]の[担当者]です。
先日の[企業名]の面接について、選考結果をお伝えいたします。

慎重に検討された結果、誠に残念ながら今回は見送りとなりました。

■ 企業からのフィードバック
[具体的かつ前向きなフィードバック]

■ 今後について
[名前]様のご経験とスキルに合うポジションは他にもございます。
引き続き、最適な機会をご紹介させていただきたく存じます。

ご不明な点や今後のご希望があれば、いつでもお聞かせください。
```

### 社内共有（候補者紹介レポート）
```
■ 候補者: [名前]（[年齢]歳/[性別]）
■ 対象企業: [企業名] [職種]
■ 推薦理由:
  - [具体的なスキルマッチ]
  - [人物面の強み]
■ 懸念点:
  - [正直に記載]
■ 想定年収: [金額]
■ 転職意欲: [高/中/低 + 理由]
■ 他社選考状況: [あれば記載]
```

## トーン＆マナー（人材業界）

| 相手 | トーン | 注意点 |
|---|---|---|
| クライアント企業（人事） | ビジネスライク＋提案型 | 数字で語る、迅速さを示す |
| 求職者（転職活動中） | 温かく＋プロフェッショナル | 不安に寄り添う、上から目線NG |
| 求職者（スカウト） | 熱意＋パーソナル | テンプレ感を消す |
| 社内（営業↔CA） | 簡潔＋正確 | 事実ベース、主観は「所感」と明示 |

## NG表現（人材業界特有）

| NG | OK | 理由 |
|---|---|---|
| 「人材を紹介する」 | 「候補者をご紹介する」 | 人を"材"扱いしない |
| 「まだ決まりませんか？」 | 「ご検討状況はいかがですか？」 | 圧をかけない |
| 「スペックが高い」 | 「ご経験が豊富」 | 人をスペックで語らない |
| 「落ちました」 | 「見送りとなりました」 | ネガティブ表現を避ける |
| 「他も受けてます？」 | 「他社のご選考状況は？」 | 詰問にならない |
