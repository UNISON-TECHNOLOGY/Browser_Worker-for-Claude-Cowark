# 日本市場の嗜好性リファレンス

公開調査・国内実務知見の再構成。web-design / video-ad / copywriting / recruit系 / sns-jp の全生成で参照する「日本で何が好まれるか」の層。

## 1. Webデザイン: 「情報の網羅性 = 信頼」

- 欧米流ミニマルは日本では「情報不足=怪しい」に転じ得る。**「ヒーローはシンプル、下層は高密度」のハイブリッド**をデフォルトに
- 日本の必須セクション（省くと信頼が落ちる）: 料金の内訳 / FAQ / 利用・購入の流れ / 運営者情報（会社概要・所在地）/ 実績数・取引先ロゴ・第三者認証
- 方針は「情報を削る」ではなく「**整理して全部載せる**」。整理の質（余白・階層）は Refactoring UI 基準を維持
- 「ちゃんとした会社」シグナル: 網羅性 + 丁寧な文章 + 顔が見える（人物写真・キャラクター）

## 2. トーン: 「丁寧でやさしい言葉」が基調

- 挑発・煽り・他社攻撃は好まれない。共感・世界観・耳ざわりのよさが現在の企業コピーの潮流
- キャラクター/マスコットの受容度は欧米より高く、信頼と親近感の装置として機能する
- タレント/出演者は「実際に愛用している感」が好意の決め手（広告っぽい起用より本人性）
- パーソナライズ（あなたへ型）は丁寧さの枠内でのみ。馴れ馴れしさは逆効果

## 3. 表記は独立したデザイン変数（日本語固有）

同じ語でも表記で訴求が変わる。訴求タイプに合わせて選ぶ:

| 表記 | 印象 | 使う場面 |
|---|---|---|
| 漢字多め | 誠実・堅実 | 信頼訴求（BtoB・金融・採用条件） |
| ひらがな多め | 柔らかい・親しみ | 共感訴求（toC・不安への寄り添い）※過多は信用低下 |
| カタカナ | 軽やか・新しさ | トレンド訴求（若年層・新サービス） |

- 数字・限定は有効だが、煽り語の連打（「今すぐ」「絶対」）には上限を設ける

## 4. 不安解消が行動の前提

- 日本の消費者・求職者は「失敗したくない」心理が強い。社会的証明（口コミ・利用者数・第三者評価）と具体的イメージ（利用の流れ・入社後の1日）が CVR/応募を動かす
- **採用文脈の優先順位（調査ベース）**: ①給与・条件の具体明示（重視58%で最多）→ ②仕事内容の詳細 → ③安定・協調の証拠（定着率・平均勤続・チームの描写）→ ④不安解消コンテンツ（1日の流れ・先輩の声）
- 新卒トレンドは「安定・調和」が過去10年で最高。**「成長・挑戦」一辺倒の訴求は現在の若年層とズレる**（成長訴求は転職顕在層向けに限定）

## 5. 動画の編集定数（日本仕様）

- カット間隔 1〜3秒（「間」を削る）、ループを意識した頭と尻の接続、最後に軽いオチ
- テロップは**白字+黒縁取りゴシック**が基本形。テレビ流の小さいサイドスーパーはスマホで読めず NG
- 縦型は主要情報を画面中央帯に（上下は UI で隠れる）
- 演出プリセットの使い分け: テレビ的（装飾テロップ・ナレーション主導）=信頼・年配向け / YouTube的（企画・解説）=検討層の理解促進 / TikTok的（UGC風・素・高テンポ）=若年認知
