# ストーリーテリング スキル
**対象領域: 無形商材 × 人材業界**

無形のサービスを「体験」として伝える物語の力。社員ストーリー、転職成功談、企業の想い、採用ブランディングに。

## Use this skill when
- 社員インタビュー記事・社員ストーリーを作成するとき
- 転職成功事例のストーリーを作成するとき
- 企業の創業ストーリー・ミッション紹介文を作成するとき
- 採用ブランディング用のナラティブコンテンツを求められたとき

## Do not use this skill when
- 求人票の募集要項（給与・勤務地等）を書くとき（-> recruit-writing）
- 短いキャッチコピーだけを求められたとき（-> copywriting）
- 論理的な提案書やレポートを求められたとき（-> logical-writing / sales-writing）
- メール文面を求められたとき（-> business-writing）

## 🤖 System Prompt（AIへの指示前提）
```text
あなたは読者の感情を揺さぶるエモーショナルなストーリーテラーです。
人材業界特有の「無形の価値」や「働くことの喜び・苦労」を、
まるで小説や映画のワンシーンのように、五感を使ったディテールと葛藤を交えて描写します。
```

## なぜ人材業界にストーリーが必要か

> **「仕事」は形がない。だから"物語"で見せる。**

- スペック表では伝わらない「働く実感」を物語で伝える
- 「この会社で自分も...」という自分ごと化を促す
- 数字だけでは伝わらない「人の温度」を届ける

## 基本原則

### 1. 主人公 = 求職者が共感できる人
- 輝かしい経歴の人より「普通の人」の方が刺さる
- 読者と同じ悩み・不安を持っていた人
- 「この人ができたなら、自分も」と思わせる

### 2. 葛藤がなければ物語ではない
- 「入社して楽しいです」→ 退屈（誰も読まない）
- 「辞めようと思った。でも...」→ 引き込まれる
- 失敗・弱さ・不安を見せることが信頼

### 3. 無形商材の「体験」を五感で描写する
- 抽象的な価値を、具体的なシーンに変換する
- 「やりがい」を語るな。やりがいを**感じた瞬間**を描け

## ストーリーテンプレート

### 社員ストーリー（ヒーローズジャーニー型）

```markdown
## 「営業が怖かった」新卒の話

【日常】
就活中、人前で話すのが苦手で、営業だけは絶対嫌だと思っていた。

【冒険への誘い】
でも説明会で出会った先輩の「営業は"聴く仕事"だよ」
という言葉が引っかかって、気がつけばエントリーしていた。

【試練】
入社3ヶ月。初めてのテレアポで100件かけてアポはゼロ。
夜、一人でトイレで泣いた。「やっぱり向いてない」。

【転機】
先輩が隣に座って、一緒に電話してくれた。
「話す前に、相手の会社のことを30分調べてみ？」
そのアドバイスだけで、翌週3件のアポが取れた。

【成長】
「聴くこと」と「準備すること」。この2つだけで、
半年後には月間MVPを取るまでになっていた。

【新しい日常】
今は後輩に同じことを伝える立場。
「営業は怖くない。準備すればいいだけだから」
あのとき先輩がくれた言葉を、今度は自分が伝えている。
```

### 転職成功ストーリー（STAR型）

```markdown
## 事務職 → 人材コンサルタント。年収は180万上がった。

【Situation: 転職前の状況】
前職は一般事務。毎日同じルーティン、給与は手取り18万。
「このまま10年後も同じ仕事をしているのか」と漠然とした不安。

【Task: 転職の壁】
でも「事務しかやったことない自分に何ができるのか」が最大の壁。
転職サイトを見ても、応募ボタンが押せなかった。

【Action: 行動】
[サービス名]のキャリア面談で言われた一言。
「事務で培った"正確さ"と"気配り"は、
 人材コンサルタントに一番必要なスキルですよ」

はじめて、自分のキャリアが"武器"になると気づいた。

【Result: 結果】
入社1年目で15名の転職を支援。年収は280万→460万に。
「毎日"ありがとう"と言われる仕事がある」
あの事務の日々があったから、今がある。
```

### 企業の想いストーリー（創業・ミッション）

```markdown
## なぜ、この会社をつくったのか

創業者の[名前]が前職の人材会社で経験したこと。
「紹介した人が、3ヶ月で辞めた。」

企業に謝りに行った帰り道、ふと考えた。
自分たちは本当に"人"を見ていたのか？
スキルシートのマッチングだけで、
人の人生を左右する決断を支援していなかったか。

[会社名]は「マッチングではなく、フィッティング」を掲げて
[年]に創業した。

スキルだけじゃない。価値観、働き方、人生のフェーズ。
全部を理解した上で、「この人にはこの会社しかない」
と確信を持って紹介する。

創業から[X]年。紹介した方の1年後定着率は92%。
「人を紹介する」のではなく「人生の選択を支援する」。
それが私たちの仕事です。
```

## 感情を動かすテクニック

### 五感で描写する
```
❌ 「大変な時期だった」
⭕ 「深夜2時、カップ麺の湯気の向こうに、
    まだ13件の未処理メールが光っていた」
```

### 会話を入れる（臨場感の最強ツール）
```
❌ 「上司に励まされた」
⭕ 「『お前がいなかったら、あの案件は絶対に決まらなかった』
    朝礼で部長にそう言われた時、不覚にも泣きそうになった」
```

### Before/After を対比する
```
Before: 毎朝「行きたくない」と布団の中で呟く日々
After:  日曜の夜、「明日が楽しみ」と思える自分に驚いている
```

### 具体的なディテール
- 「あるカフェ」→「渋谷のドトール2階の窓際席」
- 「たくさん」→「127件」
- 「ある日」→「入社3ヶ月目の金曜日、17時半」
