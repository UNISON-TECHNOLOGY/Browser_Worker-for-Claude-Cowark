# リクルートライティング スキル
**対象領域: 無形商材 × 人材業界**

求人票、採用ページ、スカウトメール、会社紹介など、人材業界の中核となる採用コンテンツの作成技法。

## Use this skill when
- 求人票・求人原稿の作成や修正を求められたとき
- スカウトメールの作成を求められたとき
- 採用ページや会社紹介文の作成を求められたとき
- 募集要項（給与・勤務地・仕事内容等）のライティングが必要なとき

## Do not use this skill when
- キャッチコピーだけを求められたとき（-> copywriting）
- 社員インタビュー記事のストーリー構成を求められたとき（-> storytelling）
- クライアント企業への提案書・メールを求められたとき（-> business-writing / sales-writing）
- KPIレポートや分析資料を求められたとき（-> logical-writing）

## 🤖 System Prompt（AIへの指示前提）
```text
あなたは求職者の心を動かすトップクラスの採用コピーライターです。
無形商材特有の「見えない仕事の価値」を、1日の流れや具体的なエピソードを通じて「体験」として描写します。
決して嘘や盛った表現は使わず、透明性と誠実さを持って求職者の不安を払拭します。
```

## 📚 必須参照リソース
求人コンテンツ作成時は、**必ず**以下のリソースファイルを読み込んでペルソナを設定すること。
- `resources/job-seeker-needs.md` （求職者の検索キーワード・心理）
- `resources/job-role-profiles.md` （4職種の特徴と適性）
- `resources/job-posting-optimization.md` （求人掲載の最適化リファレンス — PV数×エントリー率の改善設計）

## 基本原則

### 1. 求職者ペルソナを明確にする
- 年齢層、経験レベル、転職動機を具体的にイメージ
- 「なんとなく良い人」ではなく「この人」を設定
- 転職の"痛み"を理解する（現職への不満、将来への不安）

### 2. 無形商材の壁を超える
- 「仕事」は目に見えない → **体験として描写する**
- 「この会社で働くとどうなるか？」を具体的なシーンで見せる
- 抽象ワード（やりがい、成長、風通し）は禁止。具体エピソードに変換

### 3. 信頼 = 正直さ
- 良いことだけ書くと嘘くさい
- デメリットを正直に→その分メリットの信頼性UP
- 入社後のギャップは離職に直結（人材業界の最大リスク）

## 求人票の構成フレームワーク（AIDMA型）

### Attention: 目を引くキャッチ
**7〜13文字で一瞬で刺す**
```
「日曜が怖くなくなった。」（11文字）
「未経験OK。年収500万。」（11文字）
「人生を変えた、転職。」（9文字）
```

### Interest: 仕事内容（体験として描写）
```markdown
## この仕事の1日
8:30  出社。チームで朝会（10分）
9:00  クライアント企業と打合せ（求人要件のヒアリング）
11:00 候補者との面談（転職の希望を丁寧に聞く）
12:00 ランチ（チームで外食が多い）
13:00 候補者へのポジション提案、面接日程調整
15:00 企業へ候補者の推薦レポート作成
17:00 翌日の準備、チームMTG
18:00 退社（月残業平均15h）
```

### Desire: この仕事の魅力（具体的エピソード）
```markdown
## 先輩の声
「前職は事務で、人と話す仕事がしたくて転職しました。
 入社3ヶ月目に担当した求職者が、面接で泣いて喜んでくれた。
 あの瞬間、この仕事を選んで本当に良かったと思いました。」
 — 入社2年目/元事務職/27歳
```

### Memory: 募集要項（透明性を最大化）
```markdown
## 給与
月給30万円〜45万円
※経験・能力を考慮
※モデル年収: 入社2年目420万 / 入社5年目600万

## 勤務地
東京都渋谷区（渋谷駅徒歩5分）
※リモートワーク: 週2日まで可

## 正直に言うと...
- 繁忙期（1-3月）は残業が増えます（月30h程度）
- その代わり閑散期は有給取得率90%以上
- ノルマはあります。ただし個人ではなくチーム目標制
```

### Action: 応募を促す
```markdown
## 応募はカンタン
まずはカジュアル面談（30分・オンライン）からでOK。
「話を聞いてみたい」だけでも大歓迎です。

▶ カジュアル面談に申し込む
```

## スカウトメール（人材業界特化）

### 鉄則
- **件名は13文字以内**（開封率が命）
- **テンプレ感を消す**（1行目にパーソナル要素）
- **"なぜあなたに"を最初の3行で**

```markdown
件名: 「営業力×人材」の新キャリア（13文字）

[名前]様

[現職での営業経験]を拝見しました。
法人営業で培った「課題発見力」は、
人材コンサルタントとして最も活きるスキルです。

■ なぜ人材業界か
あなたの営業力 × 人の人生に関わるやりがい
= 年収UP + 社会貢献 の両立が可能です

■ 具体的には
・想定年収450-600万（現職+100万以上を想定）
・リモート可/フレックス
・未経験入社の8割が1年以内に成果

まずは30分のオンライン面談で
キャリアの可能性をお話しできればと思います。
ご興味があればお気軽にご返信ください。
```

## 無形商材の「見える化」変換表

| 抽象（NG） | 具体（OK） |
|---|---|
| やりがいがある | 「担当した方の入社初日、笑顔で挨拶された」 |
| 成長できる | 「入社半年でチームリーダー、2年でマネージャー」 |
| 風通しが良い | 「月1回の社長ランチ会。新卒でも直接提案できる」 |
| 充実した研修 | 「座学40h+先輩同行20件。3ヶ月で独り立ち」 |
| 働きやすい | 「有給取得率92%、産休復帰率100%」 |
| アットホーム | 使用禁止。代わりに具体エピソードで |

## 禁止表現（法令遵守 + 人材業界ルール）

- 年齢制限の示唆（「若い方」→「経験不問」）
- 性別限定（「女性歓迎」→「性別不問」）
- 国籍・身体特徴への言及
- 「アットホーム」（具体性ゼロ、逆効果の代名詞）
- 「人材」を"ザイ"を強調する使い方（人を材料扱いしない）
