# UI/UXデザイン心理スキル（日本実務ベース）

> 役割分担: 本スキルは「なぜそう置くか」の心理根拠。**具体的な数値（px・比率・色コード）の正本は references/design-evidence-jp/SKILL.md** — 両方に載っている値（44px・F字等）は design-evidence-jp を優先する。

## 視線・レイアウト判断表

| 対象 | 設計 | 根拠 |
|---|---|---|
| 記事型LP・一覧（読み物） | F字前提: 最重要情報と結論を左上〜冒頭2行、段落先頭に価値の高い語 | NN/g アイトラッキング（U-Site 日本語展開）。F字はモバイルでも健在 |
| バナー・初見広告 | Z型: 左上ロゴ→右上→左下→右下CTA | 日本実務の定石（F/Z/N整理） |
| 縦書き・和風（チラシ・和材） | N型: 右上起点 | 日本語縦書き特有 |
| 構造化できるページ | 見出し・箇条書きで構造化（F字詰め込みは妥協策にすぎない） | NN/g の本来の結論 |
| スマホCTA・フォーム要素 | 最小44px四方 + 周囲8px間隔。iOSセーフエリア（上44/下34px）考慮 | Apple HIG / WCAG / 日本の誤タップ5%以内検証 |

## 情報密度の判断（楽天型 vs シンプル型）

| ターゲット×訴求 | 密度 | 根拠 |
|---|---|---|
| ディスカウント・お得感・賑わい（toC・主婦層EC） | 高密度（チラシ的） | 情報量が「安さ・賑わい」の記号として機能（日本のチラシ文化） |
| 高価格帯・信頼・BtoB・ブランド | 低密度（余白） | 楽天型はこの文脈では「安っぽい・怪しい」に転じる |
| セール訴求で余白過多 | NG | 「お得感」が消える |

**優劣ではなくターゲット×価格帯×訴求のマッチング問題**（日本実務の収束見解）。

## 配色判断表

| 目的 | 色 | 注意 |
|---|---|---|
| 信頼が争点（金融・医療・BtoB SaaS） | 青基調 | 食品では食欲減退色。競合全社が青なら補色アクセントで差別化 |
| エネルギー・食欲・和 | 赤系 | エラー表示との混同に注意 |
| エコ・健康 | 緑系 | — |
| UI機能色 | エラー=赤/警告=黄/成功=緑/情報=青（JIS Z 9103 準拠） | **色だけに頼らない**（アイコン・文言併用。色覚多様性は日本で数百万人規模）。赤の多用は警告の信号価値を希釈 |

## フォーム・CV心理（EFO）

- フォーム到達者の離脱は約7割 — **必須項目はCV目的に直接必要なものだけ**。1項目削減は装飾全部より効く
- エラーはリアルタイム・項目の横に表示。住所自動入力・全半角自動変換
- **CTAボタン直下に最大の不安を打ち消す1行**（「※しつこい営業はありません」「※キャンセルいつでも可」= マイクロコピーで登録率向上の日本事例）。ただし守れない約束は逆効果、不安の書きすぎは新たな懸念を想起させる
- 個人情報の直前に利用目的を明示（日本人のデータ提供不安は7割超と国際的に高い）。電話番号必須化は離脱がCV増を上回りがち
- 料金・プランは3択・売りたいものを中間に（松竹梅=極端回避性）。CTAは1画面1種類。ただしBtoBの詳細検討層には選択肢圧縮が「情報不足」に映る

## 社会的証明・権威の見せ方（法規制と表裏）

| 手法 | 使い方 | 法規制ライン |
|---|---|---|
| レビュー・導入実績・利用者数 | CTA手前に配置。低評価も少数混在させる方が信頼される（購買影響82〜98%の調査） | **ステマ規制（2023〜）**: 依頼投稿はPR表記必須 |
| No.1・専門家推奨 | 調査主体・期間・母数を注記。監修は実名・資格明記 | イメージ調査型No.1は措置命令事例多数（消費者庁2024報告書） |
| 希少性（残り◯・◯日まで） | 実際に限られる場合のみ実数で。CTA近傍 | 虚偽は有利誤認+学習されて無効化 |
| 炎上リスク評価 | 批判の声=世論と過大視しない（炎上参加は1件あたり推計0.0015%）。ただしジェンダー・比較表現は事前レビュー | 即削除・全面謝罪が火に油のケースもある |

## 認知負荷・タイポグラフィ

- **処理流暢性**: わかりやすい＝好き・真実に見える。バナー・見出しは高コントラスト・短文・視認性の高いフォント。媒体横断で同一キービジュアル反復（単純接触）。例外: 高級・希少訴求は「読みにくさ＝特別感」が働く場合がある
- 本文は行間150%以上（デジタル庁基準1.5倍）・1行30〜40字以内（スマホで40字超は離脱要因）
- 漢字/ひらがなバランス: マス向けはひらがな多めで体感難度を下げる、BtoB・専門は漢字比率を上げて「硬さ・信頼」。ひらがな過多は幼稚に見える
